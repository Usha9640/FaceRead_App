How it works

A clean 5-step pipeline from raw image to labeled emotion output.

01
Load image with OpenCV
Read image from disk using cv2.imread(). Convert from BGR to RGB for display. Supports JPG, PNG, WEBP, and video frames.
import cv2
img = cv2.imread("happyboy.jpg")
img_rgb = cv2.cvtColor(img, cv2.COLOR_BGR2RGB)
02
Face detection with Haar Cascades
OpenCV's pre-trained Haar Cascade classifier locates face regions in the image, returning bounding box coordinates (x, y, w, h).
faceCascade = cv2.CascadeClassifier(
  cv2.data.haarcascades + 'haarcascade_frontalface_default.xml'
)
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
faces = faceCascade.detectMultiScale(gray, 1.1, 4)
03
Emotion classification via DeepFace
DeepFace runs a deep neural network on the detected face to output confidence scores for 7 emotion classes: happy, sad, angry, fear, surprise, disgust, neutral.
from deepface import DeepFace
predictions = DeepFace.analyze(img, enforce_detection=False)
if isinstance(predictions, list):
  predictions = predictions[0]
emotion = predictions['dominant_emotion']
04
Overlay bounding box & label
Draw a colored rectangle around each detected face and render the predicted emotion as text using cv2.putText().
for (x, y, w, h) in faces:
  cv2.rectangle(img, (x,y), (x+w, y+h), (0,0,255), 2)
cv2.putText(img, predictions['dominant_emotion'],
  (0, 50), font, 1, (255,0,0), 2, cv2.LINE_4)
05
Display or stream output
For static images: display with Matplotlib. For real-time: loop cap.read() and use cv2.imshow(), breaking the loop on 'q' keypress.
# Static image
plt.imshow(cv2.cvtColor(img, cv2.COLOR_BGR2RGB))

# Video stream
cap = cv2.VideoCapture(0)
if cv2.waitKey(2) & 0xFF == ord('q'): break
